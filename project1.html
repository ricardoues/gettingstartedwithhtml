Ricardo Rios's Notes on HTML

What is HTML

HTML stands for HyperText Markup Language. The web is made up of html documents.

Intro to HTML Tags

A tag in HTML has the following structure:
< NAME > Contents < /NAME >
Where < NAME > is the opening tag, < /NAME > is the closing tag, and Contents can be other tags or text.

Bold and Italics tags

Examples of these tags are as follow:
<b> Hello World </b> Hello World
<emp> Hello World </emp> Hello World

Link tag

For example, to make a link to google, all you need to do is type:
<a href="http://www.google.com"> Link to google </a>
Link to google

Adding images to your html document

If you want to add an image to your html document, all you need to do is type the following:
<img src="https://openclipart.org/image/2400px/svg_to_png/211418/programmer.png" alt="programmer" >

Span and div tags
